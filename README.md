# SparAkasha

`3D/` 配下に 3D プリント向けの部品データ（STL）を置いているリポジトリです。

## 使い方

- 上記 STL をスライサーに読み込み、各自のプリンタ／材料に合わせて出力してください。
- 寸法や強度・精度が必要な用途のため、試作→調整を前提としてください。
